Títulos de cabeçalho da seção - de h1 à h6.

>> Voltar ao topo <<

Os títulos em uma página web servem para descrever brevemente o tópico da seção em que ele está presente.
Os títulos podem ser implementados em seis diferentes níveis semânticos , <h1> sendo o de maior importância e o <h6> com menor.

O peso semântico de cada título deve ser levado em consideração para facilitar a leitura, acessibilidade e um melhor resultado no buscadores de pesquisa como Google e Bing

O Elemento Mark

>> Voltar ao topo <<

O Elemento <mark> é utilizado para dar destaque em um texto, sequencia ou palavra que tenha uma maior relevância.

O conteúdo em destaque ficará com um efeito similar a uma marcação feita no papel com um marca texto.
Para destacar o conteúdo desejado, basta encapsular o texto com uma tag <mark>

Exemplo:

HTML: Gostariamos de destacar <mark>este pedaço do texto</mark>

RESULTADO: Gostariamos de destacar este pedaço do texto



O Elemento para observações

>> Voltar ao topo <<

Este elemento é utilizadoo para inserir informações como direitos autorais ou textos legais.
Por padrão, ele é renderizado com um tamanho menor de fonte.

Exemplo:

HTML:
<p> Este texto foi retirado de um artigo sobre o crescimento da tecnologia no Brasil e as novas oportunidades que a área oferece para quem quer uma mudança de carreira</p>

<p><small>Autor do Artigo e endereço para acesso do mesmo.</small></p>



Resultado:

Este texto foi retirado de um artigo sobre o crescimento da tecnologia no Brasil e as novas oportunidades que a área oferece para quem quer uma mudança de carreira

Autor do Artigo e endereço para acesso do mesmo.



Tag <i> Transformando texto em itálico

>> Voltar ao topo <<

O Elemento <i> é utilizado para destacar alguma parte do texto que é relevante por algum motivo. Tonando o conteúdo encapsulado pela tag <i> em itálico, dando uma leve inclinação no texto



Exemplo:

HTML: Gostariamos de destacar <i>este pedaço do texto</i> e torna-lo itálico.

RESULTADO: Gostariamos de destacar este pedaço do texto e torna-lo itálico.
